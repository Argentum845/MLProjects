om	целое число	Номер Торнадо. По сути, идентификатор этого торнадо в этом году.
yr	целое число	Год, 1950-2022.
mo	целое число	Месяц, 1-12.
dy	целое число	День месяца, 1-31.
date	дата	Дата.
time	время	Время.
tz	характер	Часовой пояс базы данных Canonical tz .
datetime_utc	дата и время	Дата и время нормализованы по всемирному координированному времени.
st	характер	Двухбуквенное почтовое сокращение штата (DC = Вашингтон, округ Колумбия; PR = Пуэрто-Рико; VI = Виргинские острова).
stf	целое число	Государственный номер ФИПС (Федеральных стандартов обработки информации).
mag	целое число	Величина по шкале F (EF начиная с 2007 г.). Некоторые из этих значений являются оценочными (см. fc).
inj	целое число	Количество травм. При суммировании итоговых значений состояния используйте sn == 1 (см. ниже).
fat	целое число	Число погибших. При суммировании итоговых значений состояния используйте sn == 1 (см. ниже).
loss	двойной	Информация о предполагаемых потерях имущества в долларах. До 1996 года значения были сгруппированы в диапазоны. Зарегистрированная численность для таких лет является максимальной в своем диапазоне.
slat	двойной	Начальная широта в десятичных градусах.
slon	двойной	Начальная долгота в десятичных градусах.
elat	двойной	Конечная широта в десятичных градусах.
elon	двойной	Конечная долгота в десятичных градусах.
len	двойной	Длина в милях.
wid	двойной	Ширина в ярдах.
ns	целое число	Количество штатов, пострадавших от этого торнадо. 1, 2 или 3.
sn	целое число	Государственный номер для этой строки. 1 означает, что строка содержит всю информацию о треке для этого состояния, 0 означает, что есть как минимум еще одна запись для этого состояния для этого торнадо (ом + год).
f1	целое число	Код ФИПС для 1-го округа.
f2	целое число	Код ФИПС для 2-го округа.
f3	целое число	Код FIPS для 3-го округа.
f4	целое число	Код ФИПС для 4-го округа.
fc	логичный	Была ли оценена магнитная колонка?
